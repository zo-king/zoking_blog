# C 端布局微调 P23

## 目标

参考 `mok.moe` 的信息密度和三栏节奏，在保留 Hugo Theme Stack 视觉基底、现有路由和功能的前提下做有限调整，不复制其品牌、文案、组件或内容。

## 设计映射

- 顶部新增 56px 细导航，承载站点名、关于、GitHub、RSS 和暗色切换。
- 左栏保持 GitHub 头像和主菜单，增加文章、分类、总字数三个紧凑统计。
- 首页将独立大卡片改为统一文章流容器；桌面为 210px 封面加正文，移动端为全宽封面加正文。
- 右栏继续只显示标签，不引入参考站的评论和热门文章，避免功能重复。
- 卡片圆角收为 6px、阴影降低、段间距收为 24px，配色使用中性灰和低饱和绿色强调。

## 响应式边界

- `>= 1280px`：左 18%、右 22%，中栏自适应。
- `768px - 1279px`：保持主题既有栏位降级规则，图片宽度缩至 180px。
- `< 768px`：顶栏高 52px，隐藏重复的侧栏个人资料，保留菜单按钮；文章改为单列。
- 320px、390px 不允许页面级横向滚动，长标题自然换行。
