# 全栈博客系统需求

## 1. 项目目标

建设一个成熟完整的全栈博客系统：

- C 端读者侧保留 Hugo Theme Stack 的视觉与交互体验。
- B 端后台提供内容、发布、媒体、评论、权限、配置和数据管理能力。
- 后端采用 Go Gin + GORM + PostgreSQL。
- 系统支持长期演进，优先保证内容生产、发布稳定性、权限安全和前台阅读体验。

## 2. 用户角色

| 角色 | 说明 | 核心能力 |
|---|---|---|
| 访客 | 未登录读者 | 浏览文章、分类、标签、归档、搜索、评论 |
| 注册读者 | 可选登录读者 | 评论、管理个人资料、查看评论状态 |
| 作者 | 内容生产者 | 创建和编辑自己的文章、上传媒体、提交审核 |
| 编辑 | 内容审核与发布人员 | 审核文章、管理分类标签、发布内容 |
| 管理员 | 站点运营者 | 管理用户、角色、站点设置、审计日志 |
| 超级管理员 | 系统最高权限 | 系统初始化和高危操作 |

## 3. C 端读者侧需求

页面结构：

- 首页：左侧站点信息栏，中间文章列表，右侧推荐/分类/标签/归档信息。
- 文章详情页：标题、封面、元信息、正文、目录、标签、上一篇/下一篇、相关文章、评论区。
- 分类页：分类列表、分类下文章列表。
- 标签页：标签云、标签下文章列表。
- 归档页：按年份和月份聚合文章。
- 搜索页：支持标题、摘要、正文、标签检索。
- 关于页/自定义页面：由后台页面管理生成。
- 404 页：保持主题风格。
- RSS / Sitemap：用于订阅和搜索引擎收录。

交互：

- 支持暗色/亮色模式，默认跟随系统或站点配置。
- 支持文章目录滚动高亮。
- 支持代码块高亮、复制代码。
- 支持图片点击预览。
- 支持站内搜索，搜索结果高亮关键词。
- 支持分页，MVP 优先分页而不是无限滚动。
- 支持评论提交、回复、审核状态提示。
- 支持移动端适配。

SEO 与性能：

- 每篇文章支持独立 `title`、`description`、`canonical`、`cover`、`keywords`。
- 自动生成 Open Graph 和 Twitter Card。
- 图片使用懒加载和缩略图。
- 页面满足基础 Core Web Vitals 要求。

## 4. B 端后台信息架构

后台采用工作台式信息架构：

- 数据看板。
- 文章管理。
- 页面管理。
- 分类标签。
- 媒体库。
- 评论审核。
- 发布中心。
- 用户权限。
- 站点设置。
- 审计日志。
- 系统状态。

## 5. 数据看板

指标：

- 文章总数、已发布、草稿、待审核、定时发布。
- 页面浏览量、访客数、热门文章。
- 评论总数、待审核评论、垃圾评论。
- 媒体文件数量与存储占用。
- 最近发布记录。
- 最近操作日志。

交互：

- 支持今日、7 日、30 日、自定义时间筛选。
- 支持跳转到对应管理页面。
- MVP 可使用实时 SQL 聚合，P1 引入事件统计表。

## 6. 文章管理

字段：

- 标题、slug、摘要、正文 Markdown。
- 封面图、分类、标签。
- 作者、状态、可见性。
- SEO 标题、SEO 描述、关键词。
- 置顶、推荐、允许评论。
- 发布时间、定时发布时间、创建时间、更新时间。
- 草稿版本、发布版本。

状态：

- 草稿。
- 待审核。
- 已驳回。
- 待发布。
- 已发布。
- 已下线。
- 已归档。

功能：

- 新建、编辑、删除、恢复。
- Markdown 编辑器，支持实时预览。
- 自动保存草稿。
- 上传和插入媒体。
- 使用 C 端主题样式预览文章。
- 发布、下线、定时发布。
- 版本历史与回滚。
- 批量修改分类、标签、状态。
- 按状态、作者、分类、标签、发布时间筛选。
- 按标题、slug、摘要搜索。

## 7. 页面管理

用于管理关于页、友链页、项目页等非文章内容：

- 支持 Markdown 正文。
- 支持固定链接。
- 支持排序。
- 支持是否显示在导航。
- 支持 SEO 配置。
- 支持草稿、发布、下线。
- 支持预览。

## 8. 分类标签

分类：

- 名称、slug、描述、排序、父级分类。
- 启用/禁用。
- 文章数量统计。
- 删除前提示影响范围。

标签：

- 名称、slug、描述、颜色。
- 支持合并标签。
- 文章数量统计。
- 支持清理无文章标签。

## 9. 媒体库

支持类型：

- 图片：jpg、png、webp、gif、svg。
- 文档：pdf、txt、md。
- 音频和视频可配置。

功能：

- 上传、删除、重命名、复制 URL。
- 文件夹或标签式管理。
- 图片缩略图、尺寸、大小、上传人、引用次数。
- 图片压缩与 WebP 转换，P1 实现。
- 删除被引用媒体前必须提示。
- 开发期本地存储，生产期支持 S3/R2/MinIO。

## 10. 评论审核

状态：

- 待审核。
- 已通过。
- 已拒绝。
- 垃圾评论。
- 已删除。

功能：

- 评论列表、文章关联、用户信息、IP、User Agent。
- 通过、拒绝、标记垃圾、删除、恢复。
- 回复评论。
- 敏感词过滤。
- 频率限制。
- 黑名单：邮箱、IP、关键词。
- P1 支持邮件或 Webhook 通知。

## 11. 发布中心

发布中心负责内容发布总控：

- 展示待审核、待发布、定时发布、发布失败内容。
- 支持发布队列。
- 支持定时任务状态查看。
- 支持一键发布、一键下线。
- 支持发布前检查：slug 冲突、SEO 缺失、封面缺失、分类缺失、正文为空。
- 支持发布记录。
- P1 支持发布审批流。
- P2 支持多环境发布。

## 12. 站点设置

基础设置：

- 站点名称、副标题、Logo、Favicon。
- 站点域名、语言、时区。
- 首页分页数量。
- 默认作者。
- 默认封面图。

主题设置：

- 亮色/暗色默认模式。
- C 端导航菜单。
- 侧边栏社交链接。
- 右侧组件开关。
- 文章详情配置：目录、阅读时间、评论、上一篇/下一篇。

SEO 设置：

- 默认 SEO 标题、描述、关键词。
- Robots 配置。
- Sitemap 开关。
- RSS 开关。
- Analytics 脚本配置。

## 13. 用户权限

采用 RBAC：

- User。
- Role。
- Permission。
- RolePermission。
- UserRole。

权限粒度：

- 文章：查看、新建、编辑自己、编辑全部、审核、发布、删除。
- 页面：查看、新建、编辑、发布、删除。
- 分类标签：管理。
- 媒体：上传、删除、管理全部。
- 评论：审核、删除、回复。
- 用户：查看、创建、禁用、分配角色。
- 设置：查看、修改。
- 审计日志：查看。
- 系统：任务、缓存、健康检查。

## 14. 审计日志

记录：

- 登录、登出、登录失败。
- 创建、编辑、删除、发布、下线文章。
- 修改页面、分类、标签、媒体。
- 审核评论。
- 修改用户角色和权限。
- 修改站点设置。
- 高危操作前后摘要。

字段：

- 操作人、操作类型、资源类型、资源 ID。
- 请求 IP、User Agent。
- 操作时间。
- 操作结果。
- 变更摘要。
